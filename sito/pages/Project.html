MABALENS

The project

The Mabalens team in collaboration with IIT(Istituto Italiano di Tecnologia) for the GameAbility initiative has developed an interactive game with the integration of the Microsoft Hololens 2 mixed reality headset that allows to facilitate rehabilitation for people with multiple sclerosis.

GOALS

Improving the speed of information processing

The player will have to answer questions in the shortest time possible

 ​Motor rehabilitation of the arm

The player will have to move the arm in space to answer questions asked

Improving the dual task


The player must complete a cognitive task (choose the answer) and a motor task (select the answer)

THE GAM​E

The game is suitable for everyone.
The goal is to help people with multiple sclerosis but everybody can play it.

The game is a quiz on a chosen topic and diffculty, selected in the menu.
There are several topics: history, geography, sport, science, music and mathematic.

When the game starts the player will see the question and four possible answers. 
With the arm the user will have to select the answer that he considers correct. 

Why did we use Microsoft Hololens 2?

Microsoft Hololens 2 takes users into immersive and interactive experiences, thanks to advanced sensors, cameras and displays that create a realistic and engaging experience.
